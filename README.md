## vue-tiptap

Example component to use tiptap as a Vue component without being completely renderless, so you can see how to use it inside your own components or even use it right away.

## Install and Usage

```bash

# install dependencies
yarn

# develop
npm run dev

# build component
npm run build

# build demo
npm run build:demo

```
